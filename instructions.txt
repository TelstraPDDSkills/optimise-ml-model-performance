HOW TO RUN THIS NOTEBOOK
========================

Everything runs inside your web browser.
You do NOT need to install Python or any packages.

1. Open this link in any modern browser (Chrome, Edge, Firefox, Safari):

   https://omarkhan00.github.io/demo-optimize-ml-model/lab/index.html?path=Training_ML_Models_Optimise_Performance.ipynb

2. Wait a few seconds for the page to load.
   (It is starting Python inside your browser - this is normal.)

3. Run the first cell, titled "Install browser-compatible packages".
   Wait until it finishes before continuing.

4. Run the remaining cells from top to bottom, one at a time:
   - Click a cell, then press Shift + Enter to run it and move to the next one.
   - Or, from the top menu: Run > Run All Cells.

5. The first run downloads the packages into your browser (a few seconds).
   After that, everything runs quickly.

NOTE:
The PyCaret section near the end will NOT run in the browser, because it
needs libraries that do not work in WebAssembly. It is left in for reading
only. To run that section live, use a normal Python environment (e.g. Colab).
